Statistical Analysis in Prolog

Author: Chris Kendall
Date: 20 November 2023

This Prolog program, stats.pl, is designed for advanced statistical computations, focusing on linear regression, Pearson Correlation Coefficient, mean, and standard deviation. The program adeptly handles data from CSV files and performs complex statistical operations, showcasing the power of Prolog in data analysis.

Key Features

  1. CSV Data Handling: Efficiently loads data from specified columns of CSV files, accommodating both headers and non-header datasets.
  2. Linear Regression Analysis: Calculates alpha and beta parameters for establishing linear relationships between two data sets.
  3. Pearson Correlation Coefficient: Assesses the strength and direction of the relationship between two variables.
  4. Mean and Standard Deviation: Determines the average value and variability in a list of numbers.

Program Usage

  • CSV Reading: 
    load_data_column('data.csv', true, 1, List).
    This function reads data from a given column in a CSV file. The second argument specifies if the file has a header.
  • Linear Regression Parameters: 
    regressiona(X, Y, A), regressionb(X, Y, B).
    These functions calculate the alpha (A) and beta (B) parameters of linear regression.
  • Correlation Coefficient: 
    correlation(X, Y, R).
    Computes the Pearson Correlation Coefficient (R) between two sets of data.
  • Mean & Standard Deviation: 
    mean(L, M), stddev(L, S).
    These functions compute the mean (M) and standard deviation (S) for a list (L) of numbers.

Example Implementation

The program uses SAT scores and GPA data for demonstration. It reads from data1.csv, data2.csv, and sat-gpa.csv to perform statistical analyses.

Sample Code Snippet

Here is an excerpt showing the calculation of the mean:

mean(L, M) :-
    count_Sum(L, Sum, Num),
    ( Num > 0 -> M is Sum / Num ; M is 0).

count_Sum([], 0, 0).
count_Sum([Head|Tail], Sum, Num

) :-
    count_Sum(Tail, TailSum, TailNum),
    Sum is Head + TailSum,
    Num is 1 + TailNum.

This snippet exemplifies the Prolog approach to recursively summing a list and counting its elements to calculate the mean. The mean/2 predicate computes the average of a list L, and the count_Sum/3 helper predicate iteratively sums up the elements while counting them.

Correlation Coefficient Calculation

Another critical feature of this program is the calculation of the Pearson Correlation Coefficient:

correlation(X, Y, R) :-
    sum_Products(X, Y, SumXY),
    sum_List(X, SumX, N),
    sum_List(Y, SumY, _),
    sum_Square(X, SumX2),
    sum_Square(Y, SumY2),
    Numerator is (N * SumXY - SumX * SumY),
    DenominatorX is (N * SumX2 - SumX * SumX),
    DenominatorY is (N * SumY2 - SumY * SumY),
    Denominator is sqrt(DenominatorX * DenominatorY),
    R is Numerator / Denominator.

This part of the program demonstrates the use of Prolog’s arithmetic capabilities and list processing to compute the correlation coefficient, a measure of the linear relationship between two variables.

The program’s design emphasizes Prolog’s strengths in data processing, recursion, and mathematical calculation, making it a robust tool for statistical analysis in research and data science fields.
