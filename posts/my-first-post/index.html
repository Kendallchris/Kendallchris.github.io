Guessing Game

A guessing game to start coding in C++

In this post, I want to delve into a fun and interactive way that I begin coding in C++. I’ll discuss a simple guessing game that not only introduces basic programming concepts like loops and conditional statements but also gives insight into some fundamental data structures in C++.

Deep Dive into the Guessing Game

The game’s primary objective is for the player to guess a randomly generated number within a specified range. After each guess, the game provides feedback on the proximity of the guess to the target number, offering the player three attempts to get it right.

Key Aspects of the Code:

#include <iostream>
#include <cstdlib>
#include <ctime>

using std::cout;
using std::cin;
using std::endl;

int main() {
    srand(time(0));
    int magic, guess, cont = 1, maxNum;
    double close;

    while (cont == 1) {
        cout << "What is the maximum number you want?" << endl;
        cin >> maxNum;
        magic = rand() % maxNum + 1;
        cout << "Please guess a number" << endl;
        for (int curTry = 0; curTry < 3; ++curTry) {
            cout << "Attempt #" << curTry + 1 << ": ";
            cin >> guess;
            close = abs(magic - guess) / static_cast<double>(maxNum);
            if (guess == magic) {
                cout << "You win!" << endl;
                break;
            } else if (curTry < 2){
                cout << "Try again!" << endl;
                if (close < 0.25) {
                    cout << "Hot!" << endl;
                } else if (close > 0.25 && close < 0.50) {
                    cout << "Warm!" << endl;
                } else if (close > 0.50 && close < 0.75) {
                    cout << "Cold!" << endl;
                } else {
                    cout << "Freezing!" << endl;
                }
            }
        }
        if (guess != magic) {
            cout << "Loser" << endl;
        }
        cout << "Wanna go again? [0 for no]";
        cin >> cont;
    }
    return 0;
}

Understanding the Data Structures and Logic:

  1. Random Number Generation: The use of srand and rand functions, seeded with the current time, allows for dynamic and unpredictable gameplay.

  2. Control Structures: The outer while loop facilitates continuous gameplay until the player decides to stop. The inner for loop limits the number of guesses to three, introducing basic control flow concepts.

  3. User Interaction: The cin and cout objects are used for input and output operations, showcasing standard I/O operations in C++.

  4. Calculating Proximity: The close variable computes how near the player’s guess is to the actual number, illustrating the use of arithmetic operations and type casting in C++.

  5. Conditional Feedback: Based on the calculated proximity, conditional statements (if-else) provide appropriate feedback to the player, demonstrating the use of logical operators and decision-making in programming.

This simple game is great for beginners to understand the fundamentals of C++. The code involves using random number generation, loops for repeated attempts, and basic input/output to interact with the user.
