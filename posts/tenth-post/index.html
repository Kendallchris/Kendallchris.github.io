Implementing and Understanding Custom Data Structures in C++

Overview

Recently, I had the opportunity to delve into the complex world of data structures in C++. My project involved creating custom classes such as ‘VSArray’, ‘DoubleLinkedList’, and ‘SortedDoubleLinkedList’, each serving unique purposes and presenting their own set of challenges.

Goals

  1. ‘VSArray’: A dynamically resizable array class.
  2. ‘DoubleLinkedList’: A doubly linked list for efficient insertions and deletions.
  3. ‘SortedDoubleLinkedList’: An extension of DoubleLinkedList maintaining a sorted order of elements.

Key Features and Implementation

‘VSArray’ - Dynamic Resizing: Implemented an algorithm to resize the array based on a growth factor (’_delta’) when the capacity is reached. - Efficient Data Management: Designed methods for inserting, removing, and accessing elements while ensuring data integrity.

‘DoubleLinkedList’ - Bidirectional Navigation: Each node has pointers to both previous and next nodes, allowing for traversal in both directions. - Deep Copying: Ensured proper memory management in copy constructors and assignment operators to avoid shallow copying issues.

‘SortedDoubleLinkedList’ - Maintaining Sorted Order: Elements are inserted in a sorted order, leveraging the properties of a doubly linked list. - Search Efficiency: Benefitted from the sorted nature of the list for optimized search operations.

Challenges and Solutions

- Memory Management: Dealt with complex scenarios of memory allocation and deallocation, especially in copy constructors and destructors.
- Maintaining Sorted Order: Developed a robust insertion algorithm for 'SortedDoubleLinkedList' to ensure the list remains sorted after each insertion.

Learning Outcomes

- Enhanced Understanding of Data Structures: Gained deeper insights into how different data structures work under the hood.
- Improved C++ Skills: Advanced my C++ programming skills, particularly in areas of object-oriented programming and memory management.

Conclusion

This project was a significant step in my journey with C++. It reinforced my understanding of data structures and honed my problem-solving skills, especially in the context of managing complex memory operations in C++.
